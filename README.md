# My ToDo

シンプルで使いやすい、クラウド同期対応のToDoアプリです。

---

## アプリ概要

Google アカウントでログインするだけで、タスクをクラウド（Firebase）に保存・同期できます。
PC・スマホどちらからでも同じタスクを確認・編集でき、ホーム画面に追加してアプリのように使うこともできます（PWA対応）。

---

## 主な機能

| 機能 | 内容 |
|------|------|
| **Googleログイン** | Google アカウントで安全にログイン。自分だけのタスクが管理できます |
| **リアルタイム同期** | データは自動でクラウドに保存。PC・スマホで即座に反映されます |
| **タスク追加 / 編集 / 削除** | タスク名をクリックするとその場で編集できます |
| **期限設定** | タスクに締切日を設定できます。今日・明日が期限のものはハイライト表示されます |
| **カテゴリ分類** | 仕事 / 買い物 / 運動 / その他 の4カテゴリで整理できます |
| **フィルター** | カテゴリ・完了状態で絞り込み表示ができます |
| **並び替え** | 期限が近い順・遠い順・未完了優先・作成が新しい順の4種類から選べます |
| **期限通知** | 今日・明日が期限のタスクをブラウザ通知でお知らせします |
| **オフライン対応** | ネットワークが切れていても、キャッシュを使って表示を維持します |
| **PWA対応** | スマホのホーム画面に追加すると、ネイティブアプリのように使えます |

---

## 使用技術

| 技術 | 用途 |
|------|------|
| HTML / CSS / JavaScript | フロントエンド（ビルドツール不要の1ファイル構成） |
| [Firebase Firestore](https://firebase.google.com/docs/firestore) | タスクデータのクラウド保存・リアルタイム同期 |
| [Firebase Authentication](https://firebase.google.com/docs/auth) | Google アカウントによるログイン認証 |
| Service Worker | オフラインキャッシュ・PWA対応 |
| Web Notification API | 期限通知（ブラウザ通知） |

> ビルド不要。HTMLファイルを開くだけで動作します。

## 使い方

### 1. ログイン

アプリを開いて「Googleでログイン」ボタンをタップするだけです。
Googleアカウントがあれば誰でも使えます。

### 2. タスクを追加する

1. 上部の入力欄にタスク名を入力します
2. 必要に応じて「期限」と「カテゴリ」を設定します
3. 「追加」ボタンを押す（または Enter キー）で追加完了です

### 3. タスクを編集する

タスク名の部分をクリックすると編集フォームが開きます。
テキスト・期限・カテゴリを変更して「保存」を押してください。

### 4. タスクを完了にする

タスク左側のチェックボックスをクリックすると完了状態になります。
もう一度クリックすると未完了に戻せます。

### 5. スマホのホーム画面に追加する（PWA）

**Android（Chrome）の場合**
ブラウザのメニューから「ホーム画面に追加」を選択するか、アドレスバーに表示されるインストールアイコンをタップします。

**iPhone（Safari）の場合**
Safari の共有ボタン（□↑）→「ホーム画面に追加」を選択します。

## フォルダ構成

```
todo-app/
├── index.html          # アプリ本体（HTML / CSS / JS すべて入っています）
├── manifest.json       # PWA設定ファイル
├── service-worker.js   # オフライン対応・キャッシュ管理
├── icon.svg            # アプリアイコン（SVG）
├── icon-192.png        # PWAアイコン 192px
├── icon-512.png        # PWAアイコン 512px
└── generate-icons.html # アイコン生成ツール（開発用）
```
